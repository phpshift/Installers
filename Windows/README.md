# Development Guide

- Clone the project
- Make changes in code
- Install the Compiler `Install-Module -Name ps2exe -Scope CurrentUser`
- Then run `Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned`
- Open PowerShell and run `.\compile.ps1`
- Stage phpshift-v1.0.0.exe
- Commit changes with message "+ update {yyyy-mm-dd}"
- Push changes
