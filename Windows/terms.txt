PHPShift - World's First AI-Powered Full-Stack Framework

This installer installs the following components:
- Chocolatey package manager (if not already present)
- Python 3.11, installed to `C:\Python311` and added to system PATH
- XAMPP 8.1.25, including PHP 8.1.25, with PHP added to system PATH
- Git
- Visual Studio Code
- Composer
- Python packages `clight` and `phpshift` via `pip`
- A VS Code custom development profile
